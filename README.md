# Personal AI Operating System

A full-stack AI productivity application that helps users prioritize tasks and generate daily plans using Google Gemini AI.

## Features
- Task management
- AI task prioritization
- Daily planning
- Clean UI with Tailwind
- Gemini AI (free tier)

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express
- Google Gemini AI
